Citações

Usamos a Tag "q" para criar uma citação. Exemplo:

Como diria o pai de um amigo meu: a vingança nunca é plena. Mata a alma e envenena.

Citações em bloco

Usamos a tag "blockquote" para criar uma citação em bloco. Exemplo:

Segundo o pai de uma amiga minha:

A vingança nunca é plena. Mata a alma e envenena.

Abreviações

Utilizamos a tag "abbr" para descrevermos uma abreviação. Exemplo:

Estou aprendendo HTML e CSS e estou adorando!

Texto invertido

Usamos a tag "bdo" para produzirmos um texto escrito inverso. Exemplo:

O rato roeu a roupa do rei de roma
